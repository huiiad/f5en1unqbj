# SSP SDK GitHub Pages Source

This repository uses the `docs` folder for GitHub Pages.

The live documentation is available at: **https://ssp-sdk.github.io/ssp-sdk-android/**

## Privacy Policy

The SSP SDK Privacy Policy is hosted here and must be accessible via a public URL for app store compliance.

## Documentation Structure

```
docs/
├── index.html           # Landing page
├── index.md            # Documentation home
├── _config.yml         # Jekyll configuration
├── Gemfile            # Ruby dependencies
└── privacy/
    ├── index.md       # Privacy Policy
    └── developer-guide.md  # Developer Privacy Guide
```

## Deployment

Documentation is automatically deployed via GitHub Actions when changes are pushed to the `main` branch.
