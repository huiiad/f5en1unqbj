简介

SSP SDK（"我们"、"我们的"或"SSP"）是一家专注于为移动应用开发者提供广告变现解决方案的技术服务提供商。本隐私权政策旨在向您详细说明 SSP SDK 在为您提供服务过程中会收集哪些数据、收集的目的、如何使用这些数据，以及您享有的相关权利。

重要提示： 本隐私权政策是专门为集成 SSP SDK 的应用开发者（即"您"）编写的技术文档。它与您个人应用的隐私政策是独立的文件。当您的应用集成 SSP SDK 时，您有责任：

  1. 在您应用的隐私权政策中向最终用户披露 SSP SDK 的数据收集行为；
  2. 确保您获得最终用户对 SSP SDK 数据收集行为的有效同意；
  3. 遵守您所在司法管辖区关于数据保护和隐私的法律法规。

一、信息收集概述

1.1 收集方式

我们通过以下方式收集信息：

  • 自动收集：当最终用户的设备安装并运行集成 SSP SDK 的应用时，我们会自动收集相关设备和使用信息。
  • API 调用：我们会调用第三方服务（如 Google Play 服务）获取特定标识符信息。
  • 地理位置查询：我们会通过 IP 地址查询服务获取用户的地理位置信息。

二、信息使用目的

2.1 广告投放与优化

  • 广告定向：利用设备信息、地理位置和广告标识符向用户展示相关性更高的广告。
  • 广告竞价：在程序化广告竞价中提供必要的出价信息。
  • 频次控制：防止同一用户看到过多重复广告。
  • 反作弊：检测和防止无效流量和欺诈行为。

2.2 服务质量保障

  • 性能监控：分析广告加载速度、展示成功率等指标。
  • 故障排查：当出现技术问题时，收集相关数据用于诊断。
  • 统计分析：汇总广告效果数据，用于生成业务报告。

2.3 合规要求

  • 法律法规遵守：按照适用法律的要求保留必要的记录。
  • 争议解决：在必要时使用数据进行法律争议的解决。

三、信息共享

3.1 我们不会共享的信息

我们承诺：

  • 不会将广告标识符与其他可识别个人身份的信息关联。
  • 不会出售最终用户的个人信息。
  • 不会将收集的信息用于非广告投放以外的其他目的。

四、最终用户权利

4.1 限制广告追踪

最终用户可以通过以下方式限制广告追踪：

  1. 打开设备「设置」
  2. 进入「Google」→「广告」
  3. 启用「退出广告个性化」

4.2 重置广告标识符

用户可以随时重置其广告标识符：

  1. 打开设备「设置」
  2. 进入「Google」→「广告」
  3. 点击「重置广告 ID」

五、数据安全

5.1 安全措施

我们采用以下安全措施保护数据：

  • 传输加密：SDK 与服务器之间的通信使用 HTTPS 协议。
  • 访问控制：严格限制对服务器和数据的访问权限。
  • 定期审计：定期进行安全审计和风险评估。

六、数据保留

数据类型保留期限
广告事件数据24 个月

七、儿童隐私

我们不会故意收集 13 周岁以下儿童的个人信息。我们要求集成 SSP SDK 的应用开发者：

  1. 确保其应用不面向 13 周岁以下儿童。
  2. 如果应用面向儿童，应在隐私政策中明确说明。
  3. 确保获得监护人对数据收集的有效同意。

八、开发者义务

8.1 隐私政策披露

作为集成 SSP SDK 的开发者，您需要：

  1. 独立隐私政策：维护您自己的隐私权政策。
  2. SDK 数据披露：在您的隐私政策中明确说明 SSP SDK 的数据收集行为。
  3. 同意机制：确保获得最终用户对数据收集的适当同意。
  4. 权利保障：为用户提供行使隐私权利的渠道。

8.2 建议披露文本

我们的应用集成了 SSP SDK，用于广告展示和变现。在提供广告服务的过程中，SSP SDK 可能会收集以下信息：

  • 设备信息（品牌、型号、屏幕尺寸、操作系统版本）
  • 广告标识符（Google 广告 ID）
  • 网络信息（IP 地址、连接类型）
  • 地理位置信息（需用户授权）
  • 广告交互事件

SSP SDK 收集这些信息的目的是提供和改进广告服务。详情请参阅 SSP SDK 隐私权政策。
