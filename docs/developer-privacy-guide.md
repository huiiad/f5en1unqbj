# HuiiAd SDK 开发者隐私指南

## 概述

本指南旨在帮助开发者正确集成 HuiiAd SDK，并确保符合隐私保护法规的要求。

## 隐私合规清单

### 必检项

- [ ] 在应用隐私政策中披露 HuiiAd SDK 数据收集
- [ ] 实施最终用户同意机制
- [ ] 提供用户退出广告追踪的选项
- [ ] 未满 13 岁儿童应用需额外处理

### 推荐项

- [ ] 在权限请求时说明用途
- [ ] 提供应用内隐私设置
- [ ] 记录用户同意状态

## 常见问题

### Q: HuiiAd SDK 会收集哪些敏感信息？

A: HuiiAd SDK 会收集：
- 广告标识符（GAID）
- IP 地址
- 设备信息
- 位置信息（需用户授权）

### Q: 如何帮助用户退出广告追踪？

A: 引导用户前往系统设置：
- 设置 > Google > 广告 > 退出广告个性化

### Q: 是否需要向 SSP 提供用户同意证明？

A: 不需要，但我们建议开发者自行保留同意记录，以备监管机构查询。
